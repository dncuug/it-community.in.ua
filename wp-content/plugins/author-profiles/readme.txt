== Description ==

The plugin is developed to display the list of author in your sidebar 
using the Widget area. The Plugin used the tabular format system in which you can add any number of columns depending upon the sidebar width.You can exclude the some authors which you do no want to display in the sidebar widgets just by typing the author name separated by comma. eg:sam,sabir in the column.

<h4>Features</h4>
<ul>
<li>Each author has a link which links to the author post page. </li>
<li>Sort the authors by total number of posted by the author.</li>
<li>Make a competition of posting articles on website. for eg if the author 
has posted zero post its name will not appear in the list.</li>
<li>Display in tabular format.</li>
<li>If user mouse over on the link then it will display the author 
post counts.</li>
<li>Make your author list in fancy and attractive way.( <a href="http://codecanyon.net/item/a-fancy-wordpress-author-list/6135589" target="_blank">Pro Version</a> ) 
 For more information click here : http://themefantasy.com/plugin/wordpress-author-plugin-widget/ </li>
<li>Fancy author demo <a href="http://codecanyon.net/item/a-fancy-wordpress-author-list/full_screen_preview/6135589" target="_blank">link</a> </li>
<li>Hide the name and link ( Pro )</li>
</ul>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `authorprofile.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the widget sections to show the profiles author.
